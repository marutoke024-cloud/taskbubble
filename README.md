# Task Bubble — Abyssal Edition

Deep-sea task management. 思考の深海へダイブするタスク管理PWA。

タスクをリスト / バブルチャート / フローチャート / リファレンスボードの4ビューで管理できる、
単一HTMLファイルのオフライン対応PWAです。

- **LIST** — スター・タグ・完了管理つきのタスクリスト
- **BUBBLE** — タグ比重を落ちもの物理で可視化するバブルチャート
- **FLOW** — タスクからフローチャートを構築（Gemini AIによる自動生成対応）
- **BOARD** — 画像のリファレンスボード（PureRef風自動パッキング）

同期コード（合言葉）を設定すると、Firebase経由で端末間の手動同期ができます。

## 使い方

`index.html` をブラウザで開くだけ。ビルド不要・依存なし。
